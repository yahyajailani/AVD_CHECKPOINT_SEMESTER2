# Insight dari Analisis Pola dan Tren Perilaku Belanja Konsumen
Dari analisis data, mayoritas pelanggan adalah pria usia dewasa dengan rata-rata 44 tahun dan paling sering membeli produk kategori pakaian. Nilai belanja cenderung stabil di kisaran menengah dengan tingkat kepuasan yang cukup baik. Meski jarang menggunakan diskon atau langganan, banyak pelanggan tetap berbelanja secara rutin, menunjukkan adanya loyalitas dan kepercayaan terhadap produk.
